# 🌱 Taza Route — Алматы қайта өңдеу картасы

Интерактивная карта пунктов приёма вторсырья в Алматы.  
**Жасыл Алматы. Экологичное будущее.**

## ✨ Особенности

- 🗺️ **Интерактивная карта** Leaflet с кластеризацией маркеров
- 🌐 **Двуязычность** — Казахский (ҚР) + Русский (РУ)
- 📱 **Mobile-first дизай��** — glassmorphism UI, FAB стек
- 🧮 **CO₂ калькулятор** — расчет экономии углерода
- ♻️ **21 пункт приёма** — пластик, бумага, стекло, электроника, опасные отходы
- 🎮 **Gamification** — эко-очки, confetti анимации
- 📍 **Геолокация** — найди себя на карте
- 🔍 **Поиск** — быстрый поиск по названию и адресу
- 📋 **Список пунктов** — отсортированный просмотр
- 🔐 **PWA** — работает offline (Service Worker)
- ✨ **Glassmorphism** — современный дизайн с backdrop-filter

## 🚀 Использование

### Локально

```bash
# 1. Clone repo
git clone https://github.com/teustvrajc-create/taza-route.git
cd taza-route

# 2. Запусти локальный сервер (например, с Python)
```

## 🔧 Функционал

### Фильтры
Включи/выключи категории для фильтрации маркеров на карте.

### CO₂ Калькулятор
- Выбери материал и вес
- Получи примерную экономию CO₂ в кг

### Геолокация
Нажми кнопку "локация" → карта покажет твоё положение с маркером.

### Эко-очки
- +10 очков за каждый клик "Я здесь переработал"
- Confetti анимация
- Хранится в localStorage

### Поиск
Ищи по названию, адресу или принимаемым материалам.

## 🎯 i18n (KK / RU)

Все строки интерфейса переведены на казахский и русский.

```javascript
// Переключение языка
currentLang = 'kk'; // или 'ru'
applyI18n();
```

## 🔐 PWA (Offline)

- `manifest.json` — метаданные приложения
- `sw.js` — Service Worker для кэширования
- Работает offline после первого посещения
